## efctl

efctl manages the local EVE Frontier Sui development environment

### Synopsis

A fast and flexible CLI to automate the setup, deployment, and teardown of the EVE Frontier local world contracts and smart gates.

### Options

```
      --config-file string   Path to the efctl.yaml or efctl.yml configuration file (default "efctl.yaml")
      --debug                Enable verbose debug logging
  -h, --help                 help for efctl
      --no-progress          Disable the progress spinner for cleaner CI output
```
