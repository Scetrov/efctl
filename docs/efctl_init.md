## efctl init

Create an efctl.yaml configuration file

### Synopsis

Scaffold an efctl.yaml configuration file with the current recommended defaults for efctl.

```
efctl init [flags]
```

### Options

```
      --ai string   Setup AI instructions for a specific agent (copilot, claude, gemini)
      --force       Overwrite an existing config file
  -h, --help        help for init
```

### Options inherited from parent commands

```
      --config-file string   Path to the efctl.yaml or efctl.yml configuration file (default "efctl.yaml")
      --debug                Enable verbose debug logging
      --no-progress          Disable the progress spinner for cleaner CI output
```
